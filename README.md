# 概要
*　5つのLEDを左から順番に点灯させるプログラム

## 
* 2回実行
* 
* [ demo - YouTube](https)

## 環境
* Raspberry Pi 3 Model B
* ubuntu 18.04
* LED 5つ 
* 接続ピン - 26
* 抵抗 - 330[Ω]
* 実行方法
  * $ cd myled $ make $ sudo insmod myled.ko $ sudo chmod 666 /dev/myled0 $ chmod +x spin.py $ python spin.pu
